# 🤖 AI Bio Generator

A simple AI-inspired web application that generates professional, creative, and personalized bios based on user input such as name, profession, hobbies, tone, and platform.



## 🚀 Live Demo
https://ai-bio-generator-ai.vercel.app/

## 📌 Features

* ✨ Generate personalized bios instantly
* 🎯 Multiple tones (Professional, Friendly, Creative, Funny)
* 🌐 Platform-based output (LinkedIn, Instagram, Portfolio, Twitter)
* 📋 One-click copy to clipboard
* 🧹 Clear form option
* ⚡ Fast and responsive UI
* 🛡️ Fallback logic for reliability



## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript 
* Vercel (Deployment)

## 💡 How It Works

1. User enters:

   * Name
   * Profession
   * Hobbies
   * Preferred tone
   * Platform

2. A personalized bio is generated instantly.


## ⚙️ Setup Instructions (Local)

```bash
# Clone the repository
git clone https://github.com/Savithiri-S/AI-Bio-Generator.git

# Open folder
cd AI-Bio-Generator

# Run project
Open index.html in browser
```


## 🌐 Deployment

This project is deployed using **Vercel**:

1. Push code to GitHub
2. Import repository into Vercel
3. Click Deploy
4. Get live URL instantly
